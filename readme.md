\# Nome do Projeto



Plataforma simples para doação de itens entre pessoas. A ideia é permitir que usuários publiquem objetos que não utilizam mais para que outras pessoas possam receber gratuitamente.



\## 📦 Sobre o projeto



Este projeto funciona como um \*\*marketplace de doações\*\*, semelhante a um "OLX de doação", onde itens usados podem ganhar um novo dono em vez de serem descartados.



\## 🚀 Objetivo



\- Facilitar doações entre pessoas

\- Incentivar o reaproveitamento de objetos

\- Reduzir desperdício
